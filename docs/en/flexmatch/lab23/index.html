S3 Web Hosting

Leaderboard with S3 Web Hosting

Previously we made database, API to handle user data and game result with API Gateway and Lambda function. In this page, we are going to make static web site that S3 host. We can make Ranking board easily with this static web page with Lambda function that we made before.

  1. Move to S3 service and create S3 bucket for web site. Creating bucket, uncheck Block all public access to allow public object.

  1. There is web directory in the given source code. You can find main.js in this folder. Open main.js from Text editor.

  2. Edit API Endpoint in source code. Modify URL to Invoke URL of API Gateway that we made previously.

  1. Save modified main.js and upload all files in web directory to the S3 bucket that we made.

  2. Give Public read access permission to objects.

  3. Click Properties tab in bucket. Enable Static website hosting on this bucket.

  4. Put index.html on Index document and click “Save” button.

  1. When your files are uploaded, you are able to check web page on Static website hosting endpoint.

  2. You should configure CORS on API Gateway. You can check Ranking board data from web page by this endpoint.

  3. Move back to API Gateway, click “Actions” button and enable CORS option. (If it is disabled)

  4. Configuring CORS. Set Access-Control-Allow-Origin to your static website URL. (In this HoL, it is OK to configure * for your comfort)

  1. Click “Enable CORS” button.

  2. If it is completed, you are able to find OPTIONS in your resource tab.

  3. Click “Deploy API” button to deploy it prod stage.

You are able to check its data from API endpoint or this web page.

Web page made on this section will be used for ranking leaderboard like below.

Now, it is time to make Game Server! :)
